## 编辑器快捷键汇总

### 编辑

|   功能   |   Sublime Text   |         VS Code         |      Intellij       |  ⭐️  |
| -------- | ---------------- | ----------------------- | ------------------- | ---- |
| 复制     | cmd + c          | -                       | -                   |      |
| 粘贴     | cmd + v          | -                       | -                   |      |
| 替换     | cmd + alt + f    | -                       | cmd + r             |      |
| 局部搜索 | cmd + f          | -                       | -                   |      |
| 全局搜索 | cmd + shift + f  | -                       | double shift        | ⭐️   |
| 删除一行 | cmd + shift + d  | -                       | cmd + y             |      |
| 复制一行 | cmd + shift + d  | alt + shift + ↑/↓       | cmd + d             |      |
| 移动一行 | cmd + ctrl + ↑/↓ | alt + ↑/↓               | alt + shift + + ↑/↓ |      |
| 多行编辑 | cmd + shift + l  | cmd + alt + shift + ←/→ | /                   | ⭐️⭐️ |
| 多次选择 | cmd + d          | -                       | /                   |      |
